# Users Invite App

Интерактивное веб-приложение на **React** для отображения списка пользователей, динамического поиска, выбора и отправки приглашений.

---

## 🚀 Функциональность

* 📥 **Загрузка пользователей:** Получение данных с внешнего REST API (JSONPlaceholder).
* ⏳ **Skeleton Loading:** Индикация загрузки с использованием плавной анимации скелетонов (`react-content-loader`).
* 🔍 **Динамический поиск:** Фильтрация пользователей по имени и email в реальном времени.
* ➕ **Выбор пользователей:** Добавление и удаление пользователей из списка приглашённых с изменением состояния (плюс / минус).
* ✉️ **Отправка приглашений:** Экран успешной отправки с подсчётом количества выбранных пользователей и кнопкой возврата.

---

## 🛠 Технологии

* **React** (Hooks: `useState`, `useEffect`)
* **JavaScript (ES6+)** / **JSX**
* **SCSS / CSS**
* **React Content Loader**
